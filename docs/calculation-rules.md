# Reglas de calculo

- NetWin diario = `Coin In - Coin-Out + Jackpots`.
- Payout diario = `Coin-Out + Jackpots / Coin In`.
- Retencion = `NetWin / Coin In`.
- Apuesta media = `Coin In / Jugadas`.
- Coin In promedio por maquina y dia = `Coin In / unidades / dias`.
- NetWin promedio por maquina y dia = `NetWin / unidades / dias`.
- Ganancia por dia = `NetWin promedio por dia - costo diario`.

Los porcentajes agregados se calculan ponderando por Coin In. No se promedian porcentajes de terminales.
